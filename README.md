## 🚀 Project Overview / Descripción del proyecto

This repository contains my personal portfolio website, designed as a visual and professional space to showcase who I am, what I do and the technologies I work with.

Este repositorio contiene mi portfolio personal, diseñado como un espacio visual y profesional donde mostrar quién soy, qué hago y las tecnologías con las que trabajo.

The goal of this project is to have a clean, accessible and easily expandable web presence that can support my GitHub profile, LinkedIn profile and future job applications.

El objetivo de este proyecto es tener una presencia web clara, accesible y fácil de ampliar, que complemente mi perfil de GitHub, LinkedIn y futuras candidaturas profesionales.
